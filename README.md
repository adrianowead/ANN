# ANN

A simple example to learn how a artificial neural network works.

## Project Setup

```bash
$ php dump-autoload
```

## Usage

For a example of a ANN wich learn how to calculate a XOR, just run:

```bash
$ php xor.php
```

### Unit Tests

```bash
$ composer install
$ make test
```
